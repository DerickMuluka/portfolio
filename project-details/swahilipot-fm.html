Swahilipot FM

Modern radio streaming platform with live broadcasting capabilities

Completed: August 2024
Next.js, Node.js, PostgreSQL
Client: Swahilipot Hub Foundation
View Code Live Demo

Project Overview

Swahilipot FM is a modern radio streaming platform designed to bring the vibrant culture of Mombasa to a global audience. The platform features live broadcasting, podcast management, scheduled shows, and detailed audience analytics.

Live Streaming

Implemented low-latency audio streaming with Icecast server integration and adaptive bitrate for optimal performance across devices.

Podcast Management

Comprehensive podcast system with episode scheduling, RSS feed generation, and automatic distribution to major platforms.

Audience Analytics

Real-time listener statistics with geographic distribution, device breakdown, and listening duration metrics.

Show Scheduling

Advanced scheduling system for DJs and shows with automated reminders and conflict detection.

Technical Implementation

Technology Stack

Backend

  • Node.js with Express
  • Next.js API routes
  • Icecast streaming server
  • JWT Authentication

Database

  • PostgreSQL 14
  • Prisma ORM
  • Redis for caching
  • Full-text search

Frontend

  • Next.js 13 (App Router)
  • React Query
  • Tailwind CSS
  • Howler.js for audio

Challenges & Solutions

Low-Latency Streaming

Challenge: Required sub-10 second latency for live shows with reliable playback across varying network conditions.

Solution: Implemented HLS streaming with multiple quality levels and a WebSocket-based fallback for real-time updates.

Concurrent Listener Scaling

Challenge: Needed to support thousands of concurrent listeners during peak shows without service degradation.

Solution: Deployed on AWS with CloudFront CDN for audio distribution and auto-scaling for API servers.
