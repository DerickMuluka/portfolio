Attendance Tracking App

Biometric attendance system with facial recognition

Completed: July 2023
Kotlin, Android, Firebase
Client: Educational Institution
View Code Play Store

Project Overview

The Attendance Tracking App is a biometric solution designed to streamline attendance management in educational institutions. The system uses facial recognition technology to accurately record student attendance while preventing proxy attendance.

Facial Recognition

Advanced face detection and matching algorithms for accurate identification.

Geofencing

Location-based attendance with configurable geofence boundaries.

Reporting

Comprehensive attendance reports with export to multiple formats.

Notifications

Automated alerts for absent students and attendance thresholds.

Technical Implementation

Technology Stack

Mobile App

  • Kotlin
  • Android SDK
  • Jetpack Components
  • ML Kit for Face Detection

Backend

  • Firebase Authentication
  • Firestore Database
  • Cloud Functions
  • Firebase Storage

Security

  • Biometric Encryption
  • Role-based Access
  • Audit Logging
  • Data Anonymization

Challenges & Solutions

Offline Operation

Challenge: Needed to function reliably in areas with poor internet connectivity.

Solution: Implemented robust offline-first architecture with sync-on-reconnect.

Diverse Device Performance

Challenge: Face recognition needed to work across a wide range of Android devices.

Solution: Optimized ML models with fallback to simpler algorithms for low-end devices.
